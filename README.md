# Gerenciador de Tarefas

Aplicação web simples para gerenciamento de tarefas desenvolvida com JavaScript puro.

## Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Marcar como concluída
- Remover tarefas concluídas
- Contador dinâmico
- Persistência com localStorage

## Tecnologias

- HTML
- CSS
- JavaScript

## Aprendizados

- Manipulação do DOM
- Controle de estado da interface
- Organização de funções
- Uso de localStorage
